Simulador de Compra de Motos de Motocross

Este simulador permite al usuario elegir motos de motocross, agregarlas a un carrito de compras y calcular el precio total. El sistema utiliza funciones, arrays, condicionales y ciclos de JavaScript para simular una experiencia de compra básica mediante cuadros de diálogo.

El usuario puede consultar motos disponibles, comprar varias unidades y recibir descuentos automáticos según la cantidad comprada.
